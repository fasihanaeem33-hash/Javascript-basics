## Requirements

- Node.js (version 14 or higher) installed on your system.

## How to Run

1. Open a terminal in the project directory.
2. Run the following command:

   ```
   node app.js
   ```

3. The console will display various outputs demonstrating JavaScript basics.

## Contents

- `app.js`: Main JavaScript file with practice code including:
  - Printing messages
  - Variable declarations (let, const, var)
  - Primitive data types
  - typeof operator usage
  - Arithmetic and comparison operators
  - Increment/decrement operations
  - A simple program performing sum, subtraction, multiplication, and division

## Troubleshooting

- If `node` command is not found, ensure Node.js is installed and added to your system's PATH.
- Check for any syntax errors in `app.js` by running the file and reviewing error messages.
